# Smart Irrigation & Crop Disease Detection

## Problem
Farmers over-irrigate and detect diseases late.

## Solution
AI predicts irrigation + detects leaf disease from image.

## Features
- Irrigation recommendation
- Plant stress index
- Leaf disease detection

## Run locally
pip install -r requirements.txt
streamlit run app.py
